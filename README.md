# Análise de Qualidade de Café com NIR

Este projeto implementa um pipeline de aprendizado de máquina para analisar a qualidade do café usando dados de espectroscopia no infravermelho próximo (NIR).

## Uso

1. Instale as dependências:
```bash
pip install -r requirements.txt
```

2. Execute o script principal:
```bash
python main.py
```
